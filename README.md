# Sales Data Analysis Project

## Overview

This project analyzes sales transaction data to identify key drivers of revenue and provide actionable business insights.

## Project Structure

* business_framing/
* data_analysis/

## Objectives

* Identify top-performing products
* Evaluate promotion effectiveness
* Analyze customer behavior
* Detect anomalies in sales data

## Key Results

* Top 20% products generate most revenue
* Promotions are not consistently effective
* Some transactions show negative profit
* Customer contribution is highly uneven

## Technologies

* Python (Pandas)
* SQL
* Excel

## Business Impact

The analysis helps improve:

* Product strategy
* Promotion planning
* Customer targeting
